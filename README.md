# 🦅 Bethel International School CMS

A complete Content Management System for Bethel International School in Pawing, Palo, Leyte. This system allows school administrators to manage all website content including announcements, academic programs, news, events, admissions, calendars, newsletters, and more.

## ✨ Features

### Frontend Features
-  Responsive school website design
-  Dynamic homepage with editable sections
-  **News Highlights System** - Feature important news at the top
-  Academic programs showcase
-  Special programs section
-  Admissions information with steps
-  Contact form with database storage
-  Academic calendar (PDF download + events list)
-  Newsletter archive
-  About Us page with statistics
-  Mobile-friendly navigation
-  Coming Soon page with manual/auto toggle

### Admin Features
-  Secure admin login system
-  Complete CRUD operations
-  **News Highlight System** (⭐ Featured articles)
-  Image URL management (local or external)
-  PDF upload for calendars and newsletters
-  Coming Soon mode toggle (manual + auto)
-  Contact message management
-  Publish/draft system
-  Reorder content with display order
-  Search functionality in news management
-  Statistics dashboard
