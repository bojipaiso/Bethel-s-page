                            Version 4.0
================================================================================

1. SYSTEM REQUIREMENTS
---------------------
- PHP 7.4 or higher (PHP 8.x recommended)
- MySQL 5.7 or higher / MariaDB 10.2+
- Web server (Apache/Nginx/XAMPP/WAMP)
- 150MB disk space (plus uploads)

2. INSTALLATION STEPS
---------------------
1. Extract files to your web directory (e.g., C:\xampp\htdocs\bethel-school\)
2. Create a database named `bethel_school` in phpMyAdmin.
3. Import `database/setup.sql` into that database.
4. Edit `includes/db.php` with your database credentials.
5. Set folder permissions:
   chmod 755 uploads/ uploads/calendar/ uploads/newsletters/ uploads/news/ Images/ Images/news/
6. Create the following folders if they don't exist:
   uploads/calendar, uploads/newsletters, uploads/news, Images/news
7. Access the website: http://localhost/bethel-school/
8. Admin login: http://localhost/bethel-school/admin/login.php ( username: admin, password: admin123 )

4. ADMIN FEATURES
----------------
- Manage announcements, features, hero section.
- News articles: set featured (only one), highlight (star toggle), publish/draft.
- Academics: programs with icons, special programs.
- Admissions: content sections and steps.
- Calendar: PDF uploads and event management.
- Newsletters: PDF uploads with coming soon toggle.
- About Us: mission, vision, story, core values (each with icon & description), statistics.
- Contact messages: view, read/unread, delete.

5. CRITICAL NOTES
-----------------
- The frontend uses `css/style.css` and `includes/header.php` + `includes/footer.php`.
- The admin panel uses `css/admin-style.css`.
- All code is PHP 5.6 compatible (no null coalescing operator `??`).
- Default admin password is `password`. Change it immediately after first login.
- Delete `setup.sql` and any test files from your live server.

6. TROUBLESHOOTING
------------------
- White screen: enable error reporting in `php.ini` or check logs.
- Cannot login: verify database connection and that the admin user exists.
- Images not loading: check folder permissions and file paths.
- Star toggle not working: ensure `is_highlight` column exists in `news_articles`.

For further details, refer to README.md.
